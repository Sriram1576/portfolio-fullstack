# Portfolio Color Palette - 60-30-10 Rule

## 🎨 Minimalist Dark Theme

This portfolio uses the **60-30-10 color harmony rule** for a professional, modern aesthetic.

### Color Breakdown

| Role | Color | Hex Code | Usage | Coverage |
|------|-------|----------|-------|----------|
| **Primary (60%)** | Almost White | `#F8FAFC` | Backgrounds, main content area | Dominant |
| **Secondary (30%)** | Deep Navy Blue | `#1E3A5F` | Headers, text, sections, navigation | Major accents |
| **Highlight (10%)** | Vibrant Orange | `#FF6B35` | Buttons, CTAs, interactive elements | Key focus points |

### Supporting Colors

| Color | Hex Code | Purpose |
|-------|----------|---------|
| Surface | `#FFFFFF` | Card backgrounds, elevated surfaces |
| Text Primary | `#1E3A5F` | Main text content |
| Secondary Soft | `#E0E8F0` | Subtle accents, borders |
| Success | `#10B981` | Success messages, positive feedback |
| Error | `#EF4444` | Error states, warnings |

### Why This Palette Works

✅ **High Contrast**: Navy on white ensures excellent readability  
✅ **Professional**: Clean, minimalist aesthetic perfect for portfolios  
✅ **Modern Energy**: Orange accent adds contemporary vibrancy  
✅ **Accessible**: Meets WCAG AA contrast standards  
✅ **Memorable**: Distinct, recognizable color combination  

### Implementation Details

- **Tailwind Config**: `tech.base`, `tech.secondary`, `tech.accent`
- **CSS Variables**: `--bg-base`, `--accent-secondary`, `--accent-primary`
- **Glass Effects**: Navy-based shadows, orange accent on hover
